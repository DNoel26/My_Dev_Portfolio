A Little About Me...

A somewhat unconventional professional with a background in mechanical engineering and now...full stack web development!!!

Born and raised in Trinidad & Tobago, I believe that learning is a life-long process and thus started my venture into programming and web development. Having invested several thousands of hours into learning this craft and building some challenging, intense projects...I must say...I really enjoy it! So...what is it that I can do exactly? Well, I can build from simple websites focusing on the front-end (design, UI/UX etc.), to more complex web applications such as E-Commerce stores, utilizing the back-end (databases, server-side functionalities etc.) if need be. It's definitely a lot of work, so why do I enjoy this?

The freedom of creativity, designing and making something amazing and useful for people like you - my right brain

The constant learning and challenges, applying logic to solve difficult problems - my left brain

Alien Mathvasion

This project was designed for children ages 8+ with the goal of making math fun and engaging. It was built without any frameworks, libraries or dependencies using OOP and SOC principles, with the intention of making code DRY and easier to maintain.
Showcases heavy JavaScript usage and DOM manipulation. Uses promises instead of while loops to track progress. Design is based on retro arcade Shoot-em Up games. Good luck beating the hardest difficulty!
